# sstruth

Utility functions and SS3-related helpers.

## Installation

You can install locally:

```r
# from the project root
remotes::install_local(upgrade = "never")
# or for development
devtools::load_all()
```

## Dependencies

Imports:
- FCore
- FLBRP
- FLasher
- r4ss
- ss3om

These will be pulled in automatically when installing the package.
